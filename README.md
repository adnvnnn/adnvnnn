# عدنان عبداله زاده

توسعه‌دهنده‌ی وب با تمرکز بر طراحی و پیاده‌سازی API و ساخت سایت‌های چندزبانه.

## درباره من
من یک توسعه‌دهنده‌ی فول‌استک هستم که تمرکزم روی ساخت APIهای مقیاس‌پذیر و طراحی سایت‌هایی با پشتیبانی از چند زبان (i18n) است.
